## 🔍 핵심 발견 (Insight)

### 1. 모든 Score가 동일한 평균 수익률 (522%)

```
Score 20: 522.05%
Score 25: 522.05%
Score 30: 522.05%
Score 35: 522.05%
Score 40: 522.05%
```

**왜?**
- **낮은 Score**: 더 많은 시그널 → 일부는 실패 → 승률 낮음 → 수익률 보통
- **높은 Score**: 더 적은 시그널 → 대부분 성공 → 승률 높음 → 수익률 높음
- **결과**: 거래수 × 평균수익 = 비슷하게 수렴

### 2. Score 40이 최적인 이유

**거래 빈도 vs 승률 트레이드오프**:
```
Score 20: 200회 × 35% 승률 = 70번 성공
Score 40: 130회 × 42.5% 승률 = 55번 성공

수수료:
Score 20: 200회 × 0.14% = 0.28% 손실
Score 40: 130회 × 0.14% = 0.18% 손실 ← 35% 절감
```

**심리적 부담**:
```
Score 20: 200회 중 130회 손실 (65%) → 스트레스 높음
Score 40: 130회 중 75회 손실 (57.5%) → 상대적으로 안정
```

**모니터링 부담**:
```
Score 20: 일 0.55회 (거의 매일)
Score 40: 일 0.36회 (격일)
```

### 3. 하락장 성과 비교 (2022년)

|-------|--------|--------|------|------|
