## 3. v07 전략 방향 결정

### 핵심 개선: MACD 골든크로스 추가

**v05 (기존)**:
```python
# 진입: EMA Golden Cross만
if (prev_ema12 <= prev_ema26) and (ema12 > ema26):
    return {'action': 'buy', 'fraction': 0.95}
```

**v07 (개선)**:
```python
# 진입 조건 A: EMA Golden Cross
ema_golden = (prev_ema12 <= prev_ema26) and (ema12 > ema26)

# 진입 조건 B: MACD Golden Cross (신규)
macd_golden = (prev_macd <= prev_signal) and (macd > signal)

if ema_golden or macd_golden:
    return {'action': 'buy', 'fraction': 0.95}
```

### 예상 효과

**v05 (기존)**:
- 진입 횟수: 4~5회/년
- 수익률: 94.77%

**v07 (예상)**:
- 진입 횟수: **9~14회/년** (EMA 5회 + MACD 추가 4~9회)
- 예상 수익률: **120~140%**
- 근거:
  - MACD 단독 평균 +9.03% × 9회 = +81.27% (단순 합산)
  - 중복 제거 및 청산 로직 적용 시 실제 +25~45%p 추가 예상

### 추가 고려사항

1. **EMA + MACD 중복 진입 방지**:
   - 이미 포지션 보유 중이면 추가 매수 안 함
   - 또는 분할 매수 허용 (추후 검토)

2. **청산 조건 유지**:
   - Trailing Stop: 21%
   - Stop Loss: 10%
   - 조건 변경 없음

3. **포지션 사이징**:
   - 기존과 동일: 95% 투자
   - 추후 Kelly Criterion 적용 고려
