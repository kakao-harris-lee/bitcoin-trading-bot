## 🧪 실험 과정

### Phase 1: 멀티 타임프레임 Cascade (실패)
- **시도**: DAY(70%) + minute240(10%) + minute60(7%) + minute30(5%) + minute15(3%) + minute5(2%)
- **결과**: 34.79% (v04 대비 -253.88%p) ❌
- **원인**:
  - 자본 분할로 DAY 수익 희석
  - 짧은 타임프레임(minute5/15) 과다 거래 (646+205 = 851회)
  - 승률 24~28% (수수료 손실 과다)

### Phase 2: Simplified Cascade (실패)
- **시도**: DAY(95%) + minute240(5%)
- **결과**: 38.51% (v04 대비 -250.16%p) ❌
- **원인**: 여전히 자본 분할로 DAY 효과 감소

### Phase 3: Parameter Optimization (성공)
- **시도**: v04 DAY 전략 파라미터만 최적화
- **결과**: 293.38% (v04 대비 +4.71%p) ✅
- **방법**: Optuna로 position, trailing_stop, stop_loss 탐색

## 💡 핵심 발견 사항

### 1. **단순함이 최고**
- 복잡한 멀티 타임프레임 전략 < 단순한 DAY 전략
- 레이어 추가 → 오버트레이딩 → 수수료 손실

### 2. **자본 배분의 중요성**
- 100% DAY > 70% DAY + 30% 다른 TF
- 최고 전략에 집중 투자가 정답

### 3. **파라미터 최적화의 한계**
- position_fraction: 0.95 → 0.98 (+3%p) = 약 +5% 수익 개선
- 극적인 성과 향상은 어려움 (전략 자체가 중요)

### 4. **2024년 시장 특성**
- 강한 상승장 (Buy&Hold 137.49%)
- DAY 타임프레임이 최적 (노이즈 필터링, 주요 트렌드 포착)
- 짧은 타임프레임은 불리 (변동성 증가, 수수료 누적)

### 5. **2024 → 2025 일관성**
- 2024: 293.38% (강한 상승장)
- 2025: 124.93% (약한 상승장, Buy&Hold 20%)
- **전략의 우수성 지속**: 두 해 모두 Buy&Hold 압도

## ⚠️ 실패 사례 분석

### 멀티 타임프레임 Cascade 전략이 실패한 이유

1. **이론적 가설** (틀림):
   - "DAY 매수 중(상승장) → 짧은 TF 활성화 → 추가 수익"
   - "분할 매수/매도 → 리스크 분산"
   - "Kelly Criterion → 동적 포지션 사이징"

2. **실제 결과**:
   - minute5/15: 승률 24~25%, 과다 거래 (851회)
   - 자본 분할로 DAY 효과 70% 감소
   - 복잡도 증가, 성능 저하

3. **교훈**:
   - **검증된 전략에 집중** (DAY)
   - **불필요한 복잡도 제거**
   - **백테스팅 결과 신뢰** (minute5/15는 이미 v04에서 실패)
