## 🔑 핵심 발견 사항

### 1. 장기 데이터 학습의 함정

**가설**: "장기 데이터 학습 → 범용성 확보"
**현실**: "장기 데이터 학습 → 과도한 보수화 → 기회 손실"

**증거**:
- Trailing Stop 17.52%는 대부분의 상승장에서 너무 넓음
- 2024년 상승장(+134%)에서 100%만 달성
- 48.27%p 기회 손실

### 2. 단일 파라미터의 한계

**문제**: 상승/하락/횡보 모든 시장을 하나의 파라미터로 대응 불가

**증거**:
- 2020년 (상승장): +289.73% ✅
- 2022년 (하락장): -57.81% ❌
- 2025년 (횡보장): +13.03% ❌

### 3. Stop Loss 5.40%의 양면성

**장점**: 급락 시 빠른 손절
**단점**: 하락장에서 잦은 손절 → 반복 손실

**2022년 사례**:
- 11회 거래 모두 손실
- 진입 → 5.40% 손절 → 재진입 → 손절 반복

### 4. 오버피팅은 파라미터 문제가 아님

**발견**: 장기 데이터로 학습해도 오버피팅 87.02%
**원인**: **전략 구조 자체**의 문제
- EMA/MACD Golden Cross는 시장 환경 판단 없음
- 상승장 특화 진입 조건
- 시장 전환 감지 능력 부족
