- 결과: v13 3회 → v16 2회

### 2. 7-9월 손실 회피의 부작용
**v13의 문제**:
- 7월 진입 → 9월 손절 (-9.21%)

**Trade-off**:
- v13: -9.21% + 87.25% = +78.04% (2거래 합계)
- v16: +69.99% (1거래)
- 차이: -8.05%p 손실

### 3. ADX 횡보장 감지의 효과 미미

- 총 4일 (1.1%)

- 따라서 ADX < 15 회피 로직이 큰 영향을 주지 못함

### 4. 동적 Trailing Stop의 효과
**긍정적**:
- STRONG_TREND에서 30% Trailing Stop → 7월 거래가 12월까지 생존
